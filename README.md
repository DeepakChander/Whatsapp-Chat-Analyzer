# 📊 WhatsApp Chat Analyzer 🚀

WhatsApp Chat Analyzer is a machine learning project designed to provide detailed insights into your WhatsApp conversations! Whether you're curious about your most active friends, chat patterns, or sentiment analysis, this tool helps you visualize and understand your messaging behavior with ease. 💬📈

🌟 Key Features:

📅 Chat Activity Visualization: Analyze daily, weekly, and monthly activity trends to discover when you're most active.

👤 Participant Statistics: See which friends you chat with the most and break down message contributions in group chats.

📊 Word Cloud Generation: Visualize the most frequent words and emojis used in your conversations.

🔍 Sentiment Analysis: Understand the mood of your chats with built-in sentiment analysis, powered by machine learning algorithms.

⏱️ Response Time Analysis: Check how fast you and your friends respond to each other's messages.

📁 Export Support: Upload your WhatsApp chat export (in .txt format) and get a detailed analysis.


🔧 Technologies Used:

Python 🐍

Pandas for data manipulation and analysis

Matplotlib & Seaborn for data visualization 📊

Natural Language Processing (NLP) for sentiment analysis and text processing 🌐

Machine Learning for predictive analytics 🤖

Streamlit for an interactive and user-friendly interface

📂 How to Use:

Export your WhatsApp chat in .txt format.

Upload the file to the WhatsApp Chat Analyzer.

Let the tool crunch the data and present comprehensive analytics about your chat history in a beautiful, easy-to-read format!
